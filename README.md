# LLM-Assisted COREP Own Funds Prototype

## Scope
This prototype demonstrates a minimal LLM-assisted workflow for populating the COREP Own Funds (C 01.00) template.

## Limitations
- Only CET1, AT1, Tier 2, and Total Capital are supported
- Regulatory content is a small simulated extract
- Not production-ready

## Purpose
Demonstrate:
- Regulatory interpretation
- Structured reporting
- Validation logic
- Auditability
